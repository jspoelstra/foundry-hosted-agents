# anthropic-react-frontend

Curated skills from Anthropic for building distinctive, high-quality React and modern frontend interfaces.

**Source:** https://github.com/anthropics/skills

These skills help Claude (and compatible agents) produce polished, intentional UIs instead of generic "AI slop".

## Included Skills

- **frontend-design** — The flagship skill. Commits to a bold aesthetic direction before coding. Exceptional guidance on typography, color, motion (including React Motion), spatial composition, and avoiding predictable patterns. Works great with React + Tailwind, Vite, etc.

- **web-artifacts-builder** — Specialized for building complex, multi-component interactive React applications (React 18 + TypeScript + Vite + Tailwind + shadcn/ui patterns). Ideal for dashboards, tools, and rich web experiences.

## Usage in devpack

After `apm install`, the skills are available as:
- `/skills frontend-design`
- `/skills web-artifacts-builder`

Best results when combined with project-specific rules (e.g. your own design system tokens, component library conventions).

## Why These?

These two are the most frequently praised on X/Twitter for turning Claude into an excellent React/frontend collaborator. The `frontend-design` skill in particular is considered essential by many for any visual work.

## Future

Additional React-specific or design-system skills from the Anthropics collection (or high-quality community ones) can be added on request.
